# Pomodoro 기획서

## 1. 개요

| 항목 | 내용 |
|---|---|
| 프로젝트명 | Pomodoro |
| 한 줄 소개 | 군더더기 없는 심플한 뽀모도로 타이머 |
| 컨셉 | "타이머 하나만 잘 만들자" — 기능을 덜어내고 집중에만 집중하는 미니멀 뽀모도로 앱 |
| 플랫폼 | 웹 (반응형, 모바일 우선) |

### 배경 및 목적
시중의 뽀모도로 앱은 통계, 팀 협업, 유료 결제, 화이트노이즈 등 기능이 과도하게 많아 정작 "몰입"이라는 본질에서 멀어지는 경우가 많다. 이 프로젝트는 **타이머 시작 전까지 클릭 1번**, **화면에 보이는 요소는 타이머와 최소한의 컨트롤뿐**이라는 원칙으로 만드는 초심플 뽀모도로 앱이다.

## 2. 타겟 사용자
- 공부/업무 중 짧은 몰입 사이클이 필요한 학생, 직장인
- 복잡한 생산성 툴에 피로를 느끼는 사용자
- "그냥 타이머만 켜고 싶은" 사용자

## 3. 핵심 기능 (MVP)

### 3.1 타이머
- 기본 사이클: 집중 25분 → 짧은 휴식 5분 → (4세션마다) 긴 휴식 15분
- 사이클 값은 설정에서 자유롭게 조정 가능 (5분 단위)
- 시작 / 일시정지 / 리셋 3개 버튼만 존재

### 3.2 세션 표시
- 현재 몇 번째 뽀모도로인지 원형 도트(●●●○)로 표시
- 남은 시간은 화면 중앙에 큰 숫자로 표시 (00:00)

### 3.3 알림
- 세션 종료 시 부드러운 알림음 + 브라우저 알림(권한 허용 시)
- 소리 On/Off 토글

### 3.4 기록 (최소 기능)
- 오늘 완료한 뽀모도로 개수만 심플하게 카운트 (예: "오늘 4개 완료 🍅")
- 별도 통계 페이지/그래프는 만들지 않음 (범위 밖)

## 4. 화면 구성

```
[상단]  오늘 완료: 3개
[중앙]  ●●●○
        24:59
        (집중 중)
[하단]  ⏸ 일시정지     ↺ 리셋
[우상단] ⚙ 설정 (시간 길이, 소리, 다크모드)
```

- 화면은 이 한 페이지가 전부. 라우팅/메뉴 없음.
- 설정은 모달 하나로 처리 (집중 시간, 짧은 휴식, 긴 휴식, 알림음 On/Off, 다크모드)

## 5. 디자인 컨셉

### 5.1 방향성
**"토마토처럼 따뜻하지만, 자극적이지 않게."**
뽀모도로(pomodoro=토마토)라는 어원에서 색을 가져오되, 눈이 피로한 새빨간 색 대신 톤 다운된 웜톤을 메인으로 사용한다. 전체적으로 여백을 넉넉히 두고 타이머 숫자 외엔 시선을 뺏는 요소를 두지 않는다.

- **미니멀 & 플랫**: 그림자/그라데이션 최소화, 얇은 보더와 여백으로 구조 표현
- **라운드 형태**: 버튼과 카드는 큰 라운드(16~24px)로 부드러운 인상
- **타이핑 대신 숫자**: 타이머 숫자는 화면에서 가장 크고 굵은 요소 — 모노스페이스 폰트로 흔들림 없이 표시
- **상태별 색 전환**: 집중 중 / 휴식 중일 때 배경 톤이 은은하게 바뀌어 지금 무엇을 하고 있는지 직관적으로 인지

### 5.2 컬러 팔레트

| 이름 | 용도 | 라이트 모드 | 다크 모드 |
|---|---|---|---|
| Tomato (Primary) | 집중 세션 강조, 버튼 | `#E8604C` | `#FF7A63` |
| Tomato Soft | 집중 모드 배경 | `#FDEDE9` | `#2A1B18` |
| Leaf (Secondary) | 휴식 세션 강조 | `#5FA777` | `#7ED6A0` |
| Leaf Soft | 휴식 모드 배경 | `#EAF5EE` | `#132018` |
| Cream (Base) | 기본 배경 | `#FFFBF7` | `#1B1816` |
| Ink (Text) | 본문 텍스트 | `#2B2521` | `#F3ECE6` |
| Ink Muted | 보조 텍스트 | `#8A8078` | `#9C948C` |
| Line | 보더/구분선 | `#EFE6DE` | `#33302C` |

- 집중 세션: 배경 `Tomato Soft` + 포인트 `Tomato`
- 휴식 세션: 배경 `Leaf Soft` + 포인트 `Leaf`
- 세션 전환 시 배경색이 0.6s ease 트랜지션으로 부드럽게 교차

### 5.3 타이포그래피
- 숫자(타이머): `JetBrains Mono` / `Roboto Mono` 등 모노스페이스, Bold, 화면 폭의 30~40%
- 본문/라벨: `Pretendard` (한글) / `Inter` (영문), Regular~Medium
- 자간은 넓지 않게, 크기 대비로만 위계를 표현 (색상 대비 최소화)

### 5.4 인터랙션
- 버튼 hover/press 시 scale 0.97 정도의 미세한 눌림 효과만 사용
- 세션 종료 순간 도트가 채워지는 애니메이션(0.3s)으로 완료감 부여
- 로딩 스피너, 배너, 팝업 광고류는 일절 사용하지 않음

## 6. 기술 스택 (제안)
- Frontend: React + Vite, 상태 관리는 로컬 state/localStorage 수준으로 충분 (전역 스토어 불필요)
- 스타일: TailwindCSS (팔레트를 CSS 변수로 정의해 라이트/다크 토큰 관리)
- 알림음: Web Audio API 또는 짧은 mp3 1개
- 백엔드: 불필요 (전부 클라이언트 저장, 로그인 없음)

## 7. 비범위 (Out of Scope)
- 회원가입/로그인, 서버 동기화
- 팀/협업 기능
- 상세 통계 대시보드, 그래프
- 화이트노이즈, 플레이리스트 연동
- 유료 결제, 광고

## 8. 향후 확장 아이디어 (MVP 이후)
- 완료 뽀모도로 히스토리 캘린더 뷰
- PWA로 설치형 지원, 데스크톱 알림 강화
- 커스텀 배경 사운드 1~2개 옵션 추가
